Impact

A small JavaFX app written in Scala that finds the impact factor of scientific journals.

Installation Instructions

  1. Download the latest version of JRE 8 for your operating system and install it.
    • Windows
    • Mac OS X
  2. Download the latest release of Impact and extract it to a folder of your choice.
  3. Double-click app.jar to run the app.

Features

  • Get the latest impact factor for thousands of journals.
  • Accepts as many journals as you want, each on a separate line.
  • Export results to CSV for future use.
  • Relatively low memory usage (around 70 MB on Windows).
  • Works on all major operating systems (Windows, Mac OS X, Linux).
